# うちの子カルテ

ペットの具合が悪くなったとき、動物病院で「いつから？」「どのくらいの頻度で？」「前回から変わりましたか？」に落ち着いて答えるための、受診メモ生成Webアプリです。

お悩み解決サイト選手権の応募作です。

## 概要

「うちの子カルテ」は、気になる症状や変化をぽちぽち選ぶだけで、獣医さんにそのまま伝えやすい短い文章に整理します。

診察室で焦って説明が飛んでしまう、家では覚えていたのに聞かれるとうまく言えない、という飼い主の困りごとを助けるためのローカル保存ツールです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 「いつから」「頻度」「気になる症状」「前回受診からの変化」を選びます。
3. 必要なら補足を一言だけ入力します。
4. 生成された文章をコピーして、受診時のメモや家族への共有に使います。
5. 必要なメモは保存して、直近5件までこの端末で見返せます。

## 文章生成の考え方

AI APIは使っていません。

ボタンに表示する短い言葉と、文章に入れる自然な言い回しを分けています。選択内容は内部で意味キーに変換し、症状の組み合わせ、不明な項目、前回受診との比較ができない場合などをルールベースで組み立てます。

例：

```text
2〜3日前から、食欲の低下と嘔吐が気になっています。
気になる様子は毎日見られます。
前回の受診後より、悪くなっているように感じます。
補足として、朝ごはんを半分ほど残しました。
```

## 技術構成

- Vanilla HTML
- Vanilla CSS
- Vanilla JavaScript
- localStorage
- 外部CDN、外部API、ログイン、サーバー通信なし

## データ保護方針

入力データは利用者のブラウザのlocalStorageにだけ保存されます。

飼い主の氏名、住所、電話番号、メールアドレスは入力項目として扱いません。外部サーバーへの送信、分析タグ、メール送信、ログイン機能もありません。

端末を共有している場合は、ブラウザのサイトデータ削除機能を使って保存データを削除できます。

## 医療判断について

うちの子カルテは診断、治療方針の提示、緊急度判定を行いません。

生成される文章は、受診時に説明しやすくするためのメモです。気になる症状がある場合は、必ず獣医師に相談してください。
