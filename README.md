# うちの子カルテ

猫・犬の毎日の小さな変化を1分で残し、動物病院で「いつから？どんな感じ？」と聞かれたときに落ち着いて説明するためのローカル保存カルテです。

お悩み解決サイト選手権の応募作です。

## 概要

「うちの子カルテ」は、食事量、排便、気分、気になること、短いメモを日ごとに保存し、直近14日分の変化をカレンダーと受診用レポートにまとめる1ページWebアプリです。

記録を増やすこと自体ではなく、受診前に「いつもと違う」が日付と理由つきで整理されている安心感を目指しています。

## 使い方

1. `index.html`をブラウザで開きます。
2. 設定でペット名、種類、普段の食事・排便・気分を保存します。
3. 今日の記録で食事量、排便、気分、気になること、任意メモを保存します。
4. 過去カレンダーで直近14日の記録状態と「いつもと違う」理由を確認します。
5. レポート出力の「印刷ビューを開く」から、ブラウザ標準の印刷機能でPDF保存または紙印刷します。

## 技術構成

- Vanilla HTML
- Vanilla CSS
- Vanilla JavaScript
- localStorage
- `@media print` と `window.print()`
- 外部CDN、外部API、ログイン、サーバー通信なし

## 本質悩み

この作品が扱う悩みは、単に「ペットの健康記録を残したい」ではありません。

獣医さんに「いつから食べていませんか？」「下痢は何日続いていますか？」と聞かれたとき、毎日見ていたはずなのに答えられず、飼い主が罪悪感や焦りを感じてしまうことです。

うちの子カルテは、過去2週間の変化を日付、傾向、検出理由つきで整理し、飼い主が診察室で落ち着いて説明できる状態を作ります。

## データ保護方針

入力データは利用者のブラウザのlocalStorageにだけ保存されます。

飼い主の氏名、住所、電話番号、メールアドレスは入力項目として扱いません。外部サーバーへの送信、分析タグ、メール送信、ログイン機能もありません。

端末を共有している場合は、設定内の「この端末のデータを削除」から保存データを削除できます。

## 医療判断について

うちの子カルテは診断、治療方針の提示、緊急度判定を行いません。

「いつもと違う」表示は、受診時に説明しやすくするためのルールベースの整理です。気になる症状がある場合は、必ず獣医師に相談してください。
